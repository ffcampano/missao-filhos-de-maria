📋 Guia de Configuração — Supabase (faça uma vez só)

Passo 1. Acesse supabase.com → clique em "Start your project" → crie uma conta grátis com o Google

Passo 2. Clique em "New Project" → dê o nome "rifa-casa-de-maria" → escolha uma senha → clique em "Create new project"

Passo 3. Aguarde ~2 minutos o projeto ser criado

Passo 4. No menu lateral, clique em "SQL Editor" → clique em "New query" → cole o código abaixo e clique em "Run":

-- Cria a tabela de números da rifa
CREATE TABLE rifa_numeros (
  numero      INTEGER PRIMARY KEY,
  status      TEXT DEFAULT 'free',
  nome        TEXT,
  telefone    TEXT,
  updated_at  TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Insere os 500 números
INSERT INTO rifa_numeros (numero)
SELECT generate_series(1, 500);

-- Libera leitura pública (qualquer um vê os números)
ALTER TABLE rifa_numeros ENABLE ROW LEVEL SECURITY;
CREATE POLICY "leitura_publica" ON rifa_numeros FOR SELECT USING (true);
CREATE POLICY "escrita_publica" ON rifa_numeros FOR UPDATE USING (true);
CREATE POLICY "insercao_publica" ON rifa_numeros FOR INSERT WITH CHECK (true);

Passo 5. No menu lateral, clique em "Settings" → "API"

Passo 6. Copie os dois valores:

  • Project URL → cole no lugar de COLE_SUA_URL_AQUI
  • anon public (em "Project API Keys") → cole no lugar de COLE_SUA_ANON_KEY_AQUI

Passo 7. Salve o arquivo e suba no Vercel. Pronto — banco na nuvem funcionando! 🎉
